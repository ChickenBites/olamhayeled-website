# הוראות להוספת תמונות לאתר

## איך להוסיף תמונות חדשות:

### שלב 1: הכנת התמונות
1. העתק את התמונות החדשות לתיקיית `img` במחשב
2. **חשוב:** שנה את שמות הקבצים למספרים:
   - התמונה הראשונה: `1.jpeg`
   - התמונה השנייה: `2.jpeg`
   - התמונה השלישית: `3.jpeg`
   - וכך הלאה...

### שלב 2: העלאה לאינטרנט (ב-Windows)
פשוט לחץ פעמיים על הקובץ `add-images.bat` שנמצא בתיקייה

### שלב 3: המתן לעדכון
- העדכון יקח כמה דקות
- כתובת האתר: **https://olamhayeled.netlify.app**

---

## למה צריך למספר את התמונות?
התוכנה בודקת אוטומטית כל הזמן אם יש תמונות 1, 2, 3... בתיקייה
אז כל עוד התמונות ממוספרות בסדר, הן יופיעו באתר!

## בעיות נפוצות:
- **אין Git?** - צריך להתקין מכאן: https://git-scm.com
- **לא עובד?** - פתח את הקובץ `add-images.bat` בעורך טקסט ובדוק שכתובת ה-Git נכונה

# English Instructions:

## How to add new images:

### Step 1: Prepare images
1. Copy new images to the `img` folder
2. **Important:** Rename files to numbers:
   - First image: `1.jpeg`
   - Second image: `2.jpeg`
   - Third image: `3.jpeg`
   - And so on...

### Step 2: Upload to website (Windows)
Simply double-click `add-images.bat`

### Step 3: Wait for update
- Takes a few minutes
- Website: **https://olamhayeled.netlify.app**
